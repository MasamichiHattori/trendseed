# TrendSeed 利用規約

最終更新日：2026年6月

本利用規約（以下「本規約」）は、TrendSeed（以下「本アプリ」）の利用条件を定めるものです。本アプリを利用することで、本規約に同意したものとみなします。

## 1. サービス内容

本アプリは、海外で話題になっているショート動画の分析結果や、動画制作に活用できる情報を提供するアプリです。  
本アプリで提供される分析結果、説明、制作プロンプトなどは、動画制作の参考情報として提供されるものです。

## 2. 利用料金

本アプリは無料で利用できますが、一部機能は有料のプレミアムプランで提供されます。  
プレミアムプランでは、分析結果の閲覧制限解除、広告非表示、制作プロンプトの利用などの機能が利用できます。

## 3. サブスクリプション

本アプリのサブスクリプションはApple IDを通じて購入されます。  
サブスクリプションの管理、解約、返金については、Appleの規約およびApp Storeの設定に従うものとします。

利用者は、本アプリで提供される情報を自己の判断と責任において利用するものとします。
