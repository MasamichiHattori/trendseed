# TrendSeed サポート

TrendSeedをご利用いただきありがとうございます。  
TrendSeedは、海外で話題になっているショート動画の分析結果を閲覧し、動画制作のアイデア作成に活用できるアプリです。

## お問い合わせ

不具合、課金、機能に関するお問い合わせは、以下のメールアドレスまでご連絡ください。  
メールアドレス：support_trendseed@whai.jp

お問い合わせの際は、以下の内容を記載していただくと確認がスムーズです。

- ご利用端末
- iOSバージョン
- 発生している問題の内容
- 問題が発生した画面
- 可能であればスクリーンショット

## アプリ内課金について

TrendSeedでは、AppleのStoreKitを利用してサブスクリプションを提供しています。  
購入、解約、返金などはApple IDの設定またはApp Storeから管理できます。
